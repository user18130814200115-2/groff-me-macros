# groff-me-macros
My groff macro file which is appended at the beginning of each of my groff documents, made for use with groff-me

.bu "    "
like this
```
  - like this
```

.uh "heading title" 7 >
```
Heading title in size  18-7
```

.tt "title" "subtitle"
```
///centered
Title
Subtitle
```

.ta "author" "sub-author"
```
//centered
Author
sub-author
```
